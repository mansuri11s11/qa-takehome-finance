# QA Take-Home Assignment

## Application Chosen
OrangeHRM 
URL: https://opensource-demo.orangehrmlive.com/

Reason:
OrangeHRM provides realistic login and employee-management workflows commonly used in HR/payroll systems.

## Framework & Language
- UI Automation: Selenium + Java + TestNG
- Design Pattern: Page Object Model (POM)
- API Automation: Rest Assured + Java
- Build Tool: Maven

## How to Run
1. Clone repository
2. Run: mvn clean test
3. UI tests execute through TestNG
4. API tests execute through TestNG/JUnit

## Assumptions
- Demo environment is available.
- Test data remains unchanged.
- API examples use Reqres public API.

## Limitations
- Demo application data may reset periodically.
